Lumen V1
Local-first AI assistant built for thinking, reasoning, and building ideas.

 Overview
Lumen is a modular desktop AI assistant designed to help you learn, solve problems, design systems, and explore ideas across multiple domains.
It maintains conversational context, explains how and why, and adapts its tone and depth based on how you want to interact with it.

Math System (Dual-Layer)
Lumen handles math in two ways:
1. Direct Solving
routes to math tools
returns correct results
explains step-by-step

2. Reasoning Follow-Ups
handles questions like:
“why?”
“how did you get that?”
walks through the logic instead of repeating the answer

Conversation Continuity
Lumen maintains context across messages and supports natural follow-ups like:
“yes”
“go deeper”
“break it down step-by-step”
“continue”
These stay attached to the current topic instead of resetting.
